## Official current-version anchors

- Prisma ORM 7 upgrade guide: Node >=20.19, ESM packaging, `prisma-client`
  generator, required generator `output`, removed CLI flags, removed Prisma env
  vars including `PRISMA_CLI_QUERY_ENGINE_TYPE`, `PRISMA_CLIENT_ENGINE_TYPE`,
  and `PRISMA_QUERY_ENGINE_BINARY`:
  <https://www.prisma.io/docs/orm/more/upgrade-guides/upgrading-versions/upgrading-to-prisma-7>
- Prisma engines internals: Prisma ORM 7 defaults to the TypeScript-based query
  compiler without Rust engine binaries; `PRISMA_CLIENT_ENGINE_TYPE` and
  `PRISMA_CLI_QUERY_ENGINE_TYPE` are not supported in v7 default mode:
  <https://docs.prisma.io/docs/v6/orm/more/internals/engines>
- Prisma metrics page: the `metrics` preview feature is removed as of Prisma ORM
  v7.0.0:
  <https://www.prisma.io/docs/v6/orm/prisma-client/observability-and-logging/metrics>
- Prisma v6 environment-variable reference: documents these env vars as v6.19
  and earlier only / removed in v7:
  <https://docs.prisma.io/docs/v6/orm/reference/environment-variables-reference>
